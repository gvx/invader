Invader

Invader is a quick and arcade-ish space stragegy game. The current version is patch-level 2.

Download:

Windows Mac LÖVE/source

You can find the source code on GitHub. Invader is available under the MIT/X11 license, so have fun with it!

The game framework LÖVE gives Invader its mojo.

Invader features the song Alliance by Gobusto, which is licensed CC-BY-SA.
